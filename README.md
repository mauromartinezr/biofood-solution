# Biofood Connect

Biofood Connect converts school cafeteria data into actionable decisions. It connects WhatsApp recharge prompts, Nequi payments through Wompi, nutrition visibility for families, and analytics for cafeteria operators so parents can recharge smarter and cafeterias can improve their product mix.
